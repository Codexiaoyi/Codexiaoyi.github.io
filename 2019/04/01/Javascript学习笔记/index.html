标签

一、变量：用来保存数据

​ 语法： var 变量名=值 例：var n=23

​ 1.变量名不能以数字开头。

​ 2.变量名只能包含数字、字母、下划线。

​ 3.区分大小写。

​ 4.变量名不能喝系统内部关键字相同。

​ 5.建议：变量名要起得有意义。

二、数据类型：

​ 1.数字类型：number 数字

​ 2.字符类型：string 要加“汉字、字母、符号”

​ 3.空类型：null 表示什么都没有

​ 4.未定义类型：undefined 表示没有定义

​ 5.布尔类型：boolean true/false

​ 6.符合数据类型： array 数组 ， object 对象

​ typeof（变量名）即查看某变量的数据类型

三、运算符：

​ 1.赋值运算符：符号【=】 表示把符号右边的数据赋值给左边的变量名

​ 2.算术运算符：符号【+，-，*，%（取余），/，++，–】

​ 3.比较运算符：符号【<,>,>=,<=,==,!=,===(全等于),!==(不全等于)】

​ ==等于：判断两个数据是否相同（仅仅是判断值）

​ ===全等于：判断两个数据是否相同（判断值和数据类型）

​ 4.逻辑运算符：符号【！，&&，||】 非与或

​ 5.复合赋值运算符：符号【+=，-=，*=，/=】

​ 6.字符串运算符：符号【+】表示连接字符串

四、函数：一段能够自动完成某些功能的代码

​ 1.定义语法：

​ function 函数名(形参){

​ 函数体；

​ }

​ 2.调用语法：

​ 函数名（实参）；

​ 3.函数名：

​ 注意：不要和系统函数以及关键字相同

​ 建议：函数名要起得有意义

​ 函数可以先定义，后调用，也可以先调用，后定义。

五、数组：

​ 1.var arr=[1,2,3]

​ 2.arr.length 获取数组的长度（要保存在一个变量里面，后便直接调用变量，消除数据冗余）

六.系统内部的函数

​ 1.parseInt：取整 parseFloat：取浮点数

​ 2.NaN：不是数字（not a number）

​ isNaN：用来做判断（is not a number）

​ 当判断的数据是数字，则结果为false；当判断的数据是字符，则结果为true

七、对象

属性和方法的一般形式：

​ 属性：对象.属性名；

​ 方法：对象.方法（）；

1.Array对象：var arr=new Array（）；创建一个Array对象

​ 属性：①length：Arr.length； 获取数组的长度

​ 方法：①concat（）连接内容或数组，组成新的数组。

​ ②join（n）加入某个字符n，用n连接数组的每项组成字符串，如果以空字符串连接，那么久不会加入 新的内容，直接把数组的每项内容连接

​ ③pop（）：删除数组的最后一项并且返回

​ ④push（）；加入新的内容返回新的长度

​ ⑤reverse（）；反转数组的内容顺序（倒序）

2.String对象：var str=new String（“xxxx”）；

​ 属性：①length：Str.length； 获取字符串的长度

​ 方法：①charAt（n）：找到索引位置上的字符

​ ②charCodeAt（n）：找到索引位置上的字符的编码值（Ascull码a=97，A=65）

​ ③indexOf（“m”）：找m在字符串中第一次出现的位置，如果没有找到就返回-1

​ ④lastIndexOf（“m”）：找m在字符串中最后一次出现的位置，如果没有找到就返回-1

​ ⑤split（“n”）：以字符n分割字符串返回一个数组，以空字符串分割会把字符串每个字符都当做数 组的一项内容，如果不存在n那么就会把整个字符串当做数组的内容。

​ ⑥substr（n，m）：截取字符串：把字符串从n的位置开始截取，截取m个

​ ⑦substring（n，m）：截取字符串：把字符串从n的位置开始截取，截取到m的位置，能取到n上的字符 ，取不到m上的字符。

​ ⑧toLowerCase（）：把字符串中的字母转换成小写形式

​ ⑨toUpperCase（）：把字符串中的字母转换成大写形式

3.Math对象：

​ ①Math.pow(n,m); 表示n的m次方

​ ②Math.abs(n)； 求n的绝对值

​ ③Math.round(n)； 求n四舍五入的值

​ ④Math.floor(n)； 求n的向下取整的值：不大于n的最大整数

​ ⑤Math.ceil(n)；向上取整

​ ⑥Math.random（n）；获得一个0到1之间的随机数，注意：能取到0但是不能取到1；

​ 求任意两个数之间的随机数：Math.floor(Math.random()*(大数-小数+1)+小数)

九、定时器

一次性定时器：window.setTimeout(“js代码”,时间 t)

执行：是在是时间t之后执行js代码【只会执行一次】

时间：以毫秒为单位

反复性定时器：windoe.setInterval(“js代码”,时间t)

执行：是每过时间t就会执行一次js代码【n次】

时间：是以毫秒为单位

清除定时器：

清除一次性定时器：window.clearTimeout(abcd);

清除反复性定时器：window.clearInterval(abcd);

注意：要想清除定时器，必须给定时器名字，否则无法清除。
